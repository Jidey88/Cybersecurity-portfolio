# SOC & NOC Hands-On Labs – Babajide Keke

This repository showcases hands-on cybersecurity and networking labs focused on real-world scenarios relevant to SOC Analyst and NOC roles.

## 🔐 Projects

### 1. Palo Alto Firewall High Availability (HA)
- Configured Active/Passive HA setup using Palo Alto VM-Series
- Implemented NAT, security policies, and failover testing
- Validated redundancy and session persistence

### 2. Site-to-Site IPsec VPN
- Built secure VPN tunnel between two internal networks
- Configured IKE Phase 1 & Phase 2
- Troubleshot connectivity and policy issues

### 3. Enterprise Network Simulation
- Designed multi-site network with routers and switches
- Configured routing (OSPF/static)
- Tested connectivity and performed troubleshooting

### 4. Azure Honeypot & SIEM (Microsoft Sentinel)
- Deployed honeypot VM to capture brute-force attacks
- Analyzed logs using KQL (Event ID 4625)
- Visualized attack sources using Sentinel dashboards

### 5. Azure Identity & Cloud Security
- Configured RBAC, MFA, Conditional Access
- Implemented network segmentation with NSGs and VNets
- Monitored logs using Azure Monitor
